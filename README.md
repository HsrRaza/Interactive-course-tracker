# Interactive Course Progress Tracker

A modern course tracking application built with React, Tailwind CSS, and Framer Motion.

Users can:

- Explore courses
- Track lesson progress
- Mark lessons as completed
- View module-wise progress
- Persist progress using localStorage

---

# Features

## Homepage

- Modern dark UI
- Responsive course grid
- Animated course cards
- React Router navigation
- Framer Motion animations

## Course Page

- Expand/collapse modules
- Mark lessons complete
- Dynamic progress bar
- Progress persistence using localStorage
- Smooth animations

# Installation

Clone the repository:

```bash
git clone <repo-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

# Packages Used

Install React Router:

```bash
npm install react-router-dom
```

Install Framer Motion:

```bash
npm install framer-motion
```

Install React Icons:

```bash
npm install react-icons
```
